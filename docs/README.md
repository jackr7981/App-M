## About BD Mariner Hub

BD Mariner Hub is a comprehensive maritime career platform designed specifically for Bangladeshi seafarers. The platform aggregates job postings from multiple sources (including Telegram groups), uses AI to parse and standardize job information, and provides a clean, mobile-optimized interface for seafarers to discover opportunities.

**Key Capabilities:**
- AI-powered job aggregation from Telegram groups
- Intelligent job field extraction (8 standardized fields)
- Document management for seafarer credentials
- Sea service history tracking
- Medical center and manning agent directories
- AI-powered career assistant
- Community forum and alumni association
